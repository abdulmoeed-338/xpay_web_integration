# integration Full-stack Node.js App

This is a simple full-stack Node.js project with:

- **Backend**: Express API (`src/server.js`)
- **Frontend**: Static HTML/JS (`public/index.html`) served by Express

## Scripts

- **start**: Run the Express server:

```bash
npm start
```

Then open `http://localhost:3000` in your browser and click **“Call API”** to hit the `/api/hello` endpoint.
